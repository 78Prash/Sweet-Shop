## Description
A Sweet Shop Management System built using Spring Boot, JSP, PostgreSQL, and Spring Security.
This project supports role-based login for Admin and User.

---

## Technologies Used
- Java
- Spring Boot
- Spring MVC
- Spring Security
- JSP
- PostgreSQL
- Hibernate / JPA
- Maven
- Git & GitHub

---

## Features
- User Registration & Login
- Role-based access (ADMIN / USER)
- Admin can add, edit, delete sweets
- Users can view and purchase sweets
- Secure authentication using Spring Security
- PostgreSQL database integration

## How to Run the Project
1. Install Java 17+
2. Install PostgreSQL
3. Create database `sweetshop`
4. Update `application.properties`
5. Run Spring Boot Application
6. Open browser:
http://localhost:8080/login
